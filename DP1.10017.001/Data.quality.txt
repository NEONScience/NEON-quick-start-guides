Using a mobile data entry application with dropdown options reduces the number of processing steps to prepare the raw data for publication. The document NEON Raw Data Ingest Workbook for TOS Digital Hemispherical Photos (NEON.DP0.10017) describes how product-specific data contraints and validation requirements are built into data entry applications. The dataQF field in each data record is a quality flag that typically describes if the record was created pre-2017 when the paper-based workflow was used instead of the mobile application.
